Introdução

Bem-vindo a uma nova experiência que nunca irás esquecer, o teu PARROT AR DRONE 2.0 obteve um novo cockpit, com um controlo melhorado e uma melhor visibilidade.



Vamos começar:

Ao abrires a página de controlo, és capaz de ficar um pouco confuso, contudo, consegues ver o vídeo que a câmera do teu drone está a obter, tens acesso a ambas (clicando apenas nos botões 'Frontal' e 'Baixo'.


Antes de fazer-mos este falcão voar, temos de o calibrar para este ter noção do nível do chão, carrega no botão 'Calibrar Gyroscope' para que este calibre o giroscópio. Em suma, coloca os eixos X, Y e Z a zero.


Com isto já podemos vê-lo a voar, fantástico, agora vem o mais importante: VAMOS FAZÊ-LO VOAR, como? Através do botãozinho 'Descolagem' a verde.


Espetáculo, já voa o teu pássaro inteligente! Reparaste que todos os botões ficaram ativados, nesta parte de ajuda só vamos trabalhar com levantar e pousar, mas mais a frente podes saber como trabalhar com o resto! Agora para o pousares, tens um botãozinho amarelo que afirma 'Aterragem'


Viva! Já pousou! Como reparaste é fácil, desafio-te a começares a dominar esta ave!

Controlo por voz


Vais evoluir ainda mais o nível!

Vais utilizar o microfone para comandar o teu Drone!

Terás de utilizar a linguagem inglesa para comandar o teu drone, não te preocupes,
em cada palavra apresento-te a forma como a deves pronunciar caso tenhas dificuldades!

Em seguida, tens aqui todos os comandos que podes fornecer ao teu drone,
todos eles estão separados por gramáticas, para uma melhor organização.


  • Movimentos Principais
  • TAKEOFF - Descolagem
  • LAND - Aterragem
  • TRIM - Calibrar horizontalmente
  • STOP - Parar onde ele estiver


  • Movimentos Principais
  • MAGNET - Magnetómetro


  • Movimentos Principais
  • FRONT - Mover para a frente
  • BACK - Mover para trás
  • LEFT - Mover para a esquerda
  • RIGHT - Mover para a direita


  • Movimentos Especiais
  • UP - Mover para cima
  • DOWN - Mover para baixo

Para ativares o reconhecimnento de voz terás de carregar neste botão


Assim que carregares nele terás um som de indicador que o mesmo está a funcionar à procura de palavras!

Podes lhe dizer o teu comando!

Quando for reconhecida uma palavra um som de sucesso irá soar e verás o teu Drone executar o teu comando!
